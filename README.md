# 📚 Flashcard Quiz App

A simple and elegant Android application for creating, 
managing, and reviewing flashcards — perfect for students 
who want to test their knowledge on the go.

## 📱 About the Project

This Flashcard Quiz App was developed as part of my 
**App Development Internship at CodeAlpha**. It is designed 
to help users create custom flashcards for studying, where 
each card has a question on the front and an answer on 
the back.

---

## ✨ Features

- ➕ **Add New Flashcards** — Create custom question and 
  answer pairs
- 👁️ **Show Answer** — Tap to reveal the answer
- ➡️⬅️ **Next / Previous Navigation** — Browse through 
  flashcards easily
- ✏️ **Edit Flashcards** — Update existing cards anytime
- 🗑️ **Delete Flashcards** — Remove cards you no longer need
- 💾 **Local Storage** — All data saved locally using SQLite
- 🎨 **Clean & Minimal UI** — Simple, distraction-free design

---

## 🛠️ Built With

- **Java** — Core programming language
- **Android Studio** — Development environment
- **SQLite** — Local database for storing flashcards
- **XML** — UI layout design
